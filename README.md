# cryptarithmetic-solver
This small script will find all posible values for a "sum" of words

```
  SEND  
+ MORE
-------
 MONEY
```

## Requirements
- Node 8.9.4
- NPM 5.6.0

## dependencies
- readline
``` npm i readline ```

## usage
Simply run:
```node index.ts```
Input your words and the script will find all posible values for the sum

## example
```
  SEND
+ MORE
-------
 MONEY
```
```
  9567
+ 1085
-------
 10652
```
